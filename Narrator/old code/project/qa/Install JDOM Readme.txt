De site over het JDOM project: http://www.jdom.org/

Maak zelf gebruik van de release binarie jdom-1.0.zip (van 9 september 2004)
en van java versie 1.4 (kan standaard geinstalleerd worden op EWI systemen)

Voor installatie dienen de volgende stappen te worden ondernomen:
1. set JAVA_HOME in the environment variables van windows (set JAVA_HOME=C:\Program Files\j2sdk1.4)
2. uitvoeren van build.bat in de directory waar jdom-1.0.zip is uitgepakt
3. toevoegen van ..\jdom-1.0\build\ aan classpath
4. ook JDOM_HOME in comProg.bat goed zetten op ..\jdom-1.0\
5. run comProg.bat voor het compileren van de demo en vervolgens runProg.bat voor het starten van de demo.

Geen gebruik gemaakt van *.jar files endergelijke en ProtoFrame.java is runnable.
